# COMPONENT: ZTABLE

## 1. VARIANTS
- `default`: Standard table.
- `interactive`: Sortable, filterable.
- `compact`: Dense data display.

## 2. STRUCTURE
- `Header`: Column titles.
- `Body`: Rows, alternating background.
- `Footer`: Summary row, pagination.

## 3. RULES
- No `border-b` on every row (use sparse dividers).
- Numbers aligned right (`font-mono`).
- Text aligned left.
- Hover row highlight.
