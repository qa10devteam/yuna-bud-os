# RESPONSIVE STRATEGY

## 1. BREAKPOINTS
- `sm`: 640px.
- `md`: 768px.
- `lg`: 1024px.
- `xl`: 1280px.

## 2. PRINCIPLES
- Mobile-first.
- Grid collapses to single column.
- Navigation becomes hamburger.
- Tables become scrollable.

## 3. RULES
- Explicit mobile collapse per section.
- No assumptions.
- Test on real devices.
