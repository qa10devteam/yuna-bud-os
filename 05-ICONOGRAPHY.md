# ICONOGRAPHY: ZIEMIA DESIGN SYSTEM

## 1. ICON LIBRARY
- **Primary:** `Phosphor Icons` (`@phosphor-icons/react`).
- **Fallback:** `Hugeicons React`.
- **Style:** Outlined, geometric, consistent stroke width.
- **Stroke Width:** `1.5px` globally.

## 2. ICON USAGE
- **Size:** 24px default, 16px small, 32px large.
- **Color:** `currentColor` (inherits parent).
- **Accessibility:** `aria-label` on all interactive icons.
- **No Hand-Rolled SVGs.** Only library icons.

## 3. SHOVEL METAPHOR ICONS
- `handle`: Module 1 (Zwiad) icon.
- `shaft`: Module 2 (Kosztorys) icon.
- `blade`: Module 3 (Decyzja) icon.
- `spade`: General earthworks icon.
- `excavator`: Heavy machinery icon.
- `document`: Tender document icon.
- `calculator`: Cost estimator icon.
- `flag`: Red flag icon.
- `check`: Success/Approved icon.
- `warning`: Risk/Warning icon.

## 4. ANIMATED ICONS
- Hover states: subtle scale (`scale-110`) or color shift.
- No infinite loop animations on icons.
