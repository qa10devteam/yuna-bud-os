# LAYOUT CONTAINERS

## 1. CONTAINER TYPES
- `page`: Full width, centered content.
- `section`: Full width, padded.
- `card`: Bordered, rounded.

## 2. STYLING
- `page`: `max-w-7xl mx-auto px-4`.
- `section`: `py-16 md:py-24`.
- `card`: `p-6 bg-white dark:bg-neutral-800`.

## 3. RULES
- No overflow.
- Consistent padding.
- Responsive behavior.
