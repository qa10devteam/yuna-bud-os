# APP SILNIK FEATURES

## 1. RED FLAGS
- List of risks.
- Type, description, impact, page.
- Click to view details.

## 2. CHARTS
- Risk distribution (Pie).
- Sensitivity analysis (Tornado).
- Cost variance (Bar).

## 3. ANALYSIS
- Probability of profit.
- Worst case scenario.
- Best case scenario.
- Recommendation.

## 4. SUMMARY
- Total hidden costs.
- Recommended reserve.
- "Go to Decision" button.
