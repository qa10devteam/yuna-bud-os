# NAMING STRATEGY: Terra.OS

## 1. PRIMARY NAME: **Terra.OS**
- **Ziemia** = Earth, Ground, Soil, Foundation.
- **OS** = Operating System. It runs on Maciek's computer. It manages the earth.
- **Tagline:** "System zarządzania Ziemią dla firm budowlanych."

## 2. ALTERNATIVE NAMES (Rejected)
- `GRUNT` → Too generic, conflicts with "ground/earth" but feels like a task.
- `KOPARKA` → Too hardware-focused.
- `ZIEMEX` → Sounds like 90s software.
- `ROBO` → Too playful.
- `KOSZTORYSANT` → Only covers Module 2.
- `ZIEMIA AI` → Redundant (AI is implied by OS).

## 3. BRAND POSITIONING
- **Tone:** Professional, technical, reliable.
- **Audience:** Maciek (owner/operator), his cost estimators, his site foremen.
- **Vibe:** Industrial efficiency. "Built for the field, designed for the office."

## 4. LOGO CONCEPT
- Abstract shovel silhouette merged with a digital node/network.
- Monochrome (white/black) with single accent color (Neon Green).
