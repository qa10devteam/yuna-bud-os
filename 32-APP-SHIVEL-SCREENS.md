# APP SHIVEL SCREENS

## 1. SCREENS
1. `ZWIAD`: Tender search, filters, list.
2. `KOSZTORYS`: Cost comparison, sliders, tables.
3. `SILNIK`: Risk analysis, red flags, charts.
4. `DECYZJA`: Summary, CTA, decision.
5. `MOZG`: Construction planning, calendar, teams.

## 2. NAVIGATION
- Vertical sidebar (shovel metaphor).
- Handle (Zwiad) → Shaft (Kosztorys/Silnik) → Blade (Decyzja/Mózg).
- Active state highlight.
- Progress indicator.

## 3. TRANSITIONS
- Smooth scroll between sections.
- Page transitions with Motion.
- Loading states.
