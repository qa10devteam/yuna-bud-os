# COMPONENT: ZCHART

## 1. LIBRARY
- `Recharts` or `Tremor`.

## 2. TYPES
- `LineChart`: Trends, costs over time.
- `BarChart`: Comparisons (document vs. your costs).
- `PieChart`: Risk distribution.
- `AreaChart`: Profit margins, probability distributions.
- `TornadoChart`: Sensitivity analysis.

## 3. STYLING
- Colors: `--color-accent-success`, `--color-accent-warning`, `--color-accent-tech`.
- Grid lines: `stroke-neutral-200`.
- Labels: `font-mono` for numbers.
- Tooltips: Dark background, light text, rounded.

## 4. RULES
- Charts must be responsive.
- Labels must be readable.
- Legends must be clear.
- No 3D charts.
