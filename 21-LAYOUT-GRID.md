# LAYOUT GRID

## 1. GRID SYSTEM
- `12-column` grid.
- `gap-6` gutter.
- `px-4` mobile padding.
- `max-w-7xl` desktop max-width.

## 2. BREAKPOINTS
- `sm`: 640px.
- `md`: 768px.
- `lg`: 1024px.
- `xl`: 1280px.
- `2xl`: 1536px.

## 3. RULES
- Grid over flex-math.
- Consistent gutter.
- Responsive collapse.
