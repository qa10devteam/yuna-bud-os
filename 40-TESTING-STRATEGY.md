# TESTING STRATEGY

## 1. UNIT TESTS
- Vitest.
- Components, utilities, stores.

## 2. INTEGRATION TESTS
- Playwright.
- User flows, navigation, forms.

## 3. PERFORMANCE
- Lighthouse.
- LCP < 2.5s.
- INP < 200ms.
- CLS < 0.1.

## 4. ACCESSIBILITY
- axe-core.
- WCAG AA compliance.

## 5. RULES
- Test every component.
- Test every flow.
- Test on mobile.
