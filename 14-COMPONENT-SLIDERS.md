# COMPONENT: ZSLIDER

## 1. VARIANTS
- `default`: Range slider.
- `number`: Slider with number input.
- `vertical`: Vertical slider (rare).

## 2. STYLING
- `track`: `bg-neutral-200`.
- `fill`: `bg-accent-tech`.
- `thumb`: `bg-white border-2 border-accent-tech`.

## 3. RULES
- Sliders must update data in real-time.
- Values must be displayed.
- Labels must be clear.
