# APP ZWIAD FEATURES

## 1. MAP
- Interactive map of Poland.
- Highlight 4 provinces (Dolnośląskie, Śląskie, Opolskie, Lubuskie).
- Click to filter tenders.

## 2. FILTERS
- Province (checkboxes).
- Type (select).
- Value (slider).
- Deadline (date picker).

## 3. LIST
- Tender cards.
- Match score badge.
- Red flags badge.
- Click to view details.

## 4. DETAIL VIEW
- Tender description.
- Documents list.
- Red flags list.
- "Go to Costing" button.
