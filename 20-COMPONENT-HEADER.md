# COMPONENT: ZHEADER

## 1. VARIANTS
- `default`: Page header.
- `hero`: Hero section header.
- `sticky`: Sticky header.

## 2. STRUCTURE
- `Title`: Page title.
- `Subtitle`: Page description.
- `Actions`: CTA buttons.

## 3. RULES
- Hero fits viewport.
- Max 2 lines headline.
- Max 20 words subtext.
- CTA visible without scroll.
