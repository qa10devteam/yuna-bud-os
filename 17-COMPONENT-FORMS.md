# COMPONENT: ZFORM

## 1. VARIANTS
- `default`: Standard form.
- `wizard`: Multi-step form.
- `inline`: Inline form (search, filters).

## 2. STRUCTURE
- `Field`: Label, Input, Helper/Error.
- `Actions`: Submit, Cancel, Reset.
- `Progress`: Step indicator (wizard).

## 3. RULES
- Validation on submit and blur.
- Error messages actionable.
- Loading state on submit.
- Success state after submit.
