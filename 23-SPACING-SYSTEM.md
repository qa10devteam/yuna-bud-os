# SPACING SYSTEM

## 1. BASE UNIT
- `4px`.

## 2. SCALE
- `space-1`: 4px.
- `space-2`: 8px.
- `space-3`: 12px.
- `space-4`: 16px.
- `space-6`: 24px.
- `space-8`: 32px.
- `space-12`: 48px.
- `space-16`: 64px.
- `space-24`: 96px.

## 3. RULES
- Use Tailwind spacing classes.
- Consistent vertical rhythm.
- No arbitrary spacing values.
