# COMPONENT: ZNAVIGATION

## 1. VARIANTS
- `default`: Top bar.
- `tab`: Tab navigation.
- `breadcrumb`: Breadcrumb navigation.

## 2. STRUCTURE
- `Logo`: Brand mark.
- `Menu`: Navigation items.
- `Actions`: Search, user, notifications.

## 3. RULES
- Single line nav items.
- Active item highlight.
- Responsive collapse on mobile.
