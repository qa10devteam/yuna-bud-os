# COMPONENT: ZSIDEMENU

## 1. VARIANTS
- `default`: Left sidebar, fixed width.
- `collapsible`: Collapsible on mobile.
- `floating`: Floating menu (rare).

## 2. STRUCTURE
- `Header`: Logo, toggle button.
- `Body`: Navigation items, grouped by section.
- `Footer`: User profile, settings.

## 3. RULES
- Max 80px height desktop nav.
- Single line nav items.
- Active item highlight.
- Keyboard accessible.
